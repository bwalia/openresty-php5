About Gebo Admin template

Gebo Admin is a fully featured, easily customizable backend template that can be use for any web aplication (cms,crm,statistics etc).

Main features:

  1. Responsive two column layout (hideable sidebar)
  2. Fixed top bar
  3. HTML5/CSS3
  4. Based on Twitter Bootstrap Framework
  5. Location finder (Google Maps)
  6. Custom search page
  7. Charts (lines, bars, pies)
  8. Datepicker, Timepicker
  9. Image grid
  10. 2 icon sets (glyphicons & splashy icons)
  11. Custom error page
  12. Tables (static, dynamic, server side proccessing)
  13. Notifications (sticky, modal boxes)
  14. Responsive file explorer
  15. WYSIWYG editor with integrated file explorer
  16. Regular and extended form elements (sliders, progresbars, enchanced select elements, 2 column multiselect, styled form elements, masked inputs, tags handler and many more)
  17. Tooltips (always visible in the viewport), Popovers
  18. Bootstrap Javascript Plugins (modal, dropdown, tab, popover, collapse, typeahead, rowlink, fileupload )
  19. Form Validation
  20. Step by Step Wizard

Main structure

Gebo uses a layout from Bootstrap Framework. This layout is fully responsive, it looks good on desktop, tables and mobile devices. It has top fixed bar (collapsible on mobile devices), fluid content and sidebar with search form and collapsible menu.

Sidebar

Gebo Admin has two columns by default, main content and a hideable sidebar. This sidebar can be hidden on page load by adding sidebar_hidden class to body. In this template sidebar state is saved to cookie. If you don't want this functionality just coment/remove some code in js/gebo_common.js (lines: 50-62;70;75).

Layout, grid & elements

Gebo Admin is based on Bootstrap Framework, they have great documetation on http://twitter.github.com/bootstrap/index.html. If you need to find out how elements works, how to build custom layout or how to use their js plugins please use Bootstrap doumentation.
